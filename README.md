# рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃфрЃћрЃЏрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА API

RESTful API рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃфрЃћрЃЏрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃАрЃљрЃЊрЃљрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА рЃерЃћрЃћрЃФрЃџрЃћрЃЉрЃљрЃЌ рЃБрЃцрЃљрЃАрЃЮрЃЊ рЃњрЃљрЃАрЃфрЃћрЃю, рЃљрЃАрЃћрЃЋрЃћ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃАрЃњрЃљрЃю рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃЮрЃю рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃгрЃўрЃњрЃюрЃћрЃЉрЃў.

## рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў
- ­ЪЊџ **рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ**: рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ, рЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃгрЃљрЃерЃџрЃљ (CRUD)
- ­ЪЉЦ **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**: рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ, рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ, рЃърЃарЃЮрЃцрЃўрЃџрЃћрЃЉрЃў
- ­ЪћЇ **рЃФрЃћрЃЉрЃюрЃљ рЃЊрЃљ рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ**: рЃФрЃўрЃћрЃЉрЃљ рЃЊрЃљ рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃА, рЃЪрЃљрЃюрЃарЃўрЃА, рЃгрЃџрЃўрЃА, рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ
- ­ЪЊІ **рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**: рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
- РГљ **рЃарЃћрЃўрЃбрЃўрЃюрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃфрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ
- ­ЪЊ▒ **API рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ**: Swagger рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў

## рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў
- **Backend**: Python 3.11, Django 4.2, Django REST Framework
- **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ**: SQLite
- **рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃўрЃќрЃљрЃфрЃўрЃљ**: Docker, Docker Compose
- **рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ**: Swagger (drf-yasg)
- **рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ**: Token-based authentication

## рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ

### рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў
- Python 3.11+
- Docker лИ Docker Compose (рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃўрЃљ)

### рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

#### Docker-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ
**рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃўрЃА рЃЊрЃљрЃЎрЃџрЃЮрЃюрЃЋрЃљ рЃЊрЃљ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ**
```bash
# рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃўрЃА рЃЊрЃљрЃЎрЃџрЃЮрЃюрЃЋрЃљ
git clone https://github.com/vakho-jin/book-sharing.git
# рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃерЃћрЃАрЃЋрЃџрЃљ
cd book-sharing
# рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
python -m venv venv
#рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ
source venv/bin/activate  # Linux/Mac
# рЃљрЃю
venv\Scripts\activate  # Windows
# рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ
pip install -r requirements.txt
# рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ
python manage.py makemigrations
python manage.py migrate
# рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
python manage.py create_conditions
python manage.py load_initial_data
# рЃАрЃБрЃърЃћрЃарЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
python manage.py createsuperuser
# рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ
python manage.py runserver
```

## API Endpoints

### рЃљрЃЋрЃарЃбрЃЮрЃарЃўрЃљрЃќрЃљрЃфрЃўрЃљ
 - POST /api/auth/register/ - рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ
 - POST /api/auth/login/ - рЃерЃћрЃАрЃЋрЃџрЃљ
 - POST /api/auth/logout/ - рЃњрЃљрЃЏрЃЮрЃАрЃЋрЃџрЃљ
 - GET/PUT /api/auth/profile/ - рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ/рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ

### рЃгрЃўрЃњрЃюрЃћрЃЉрЃў
 - GET /api/books/ - рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ
 - POST /api/books/ - рЃљрЃ«рЃљрЃџрЃў рЃгрЃўрЃњрЃюрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
 - GET /api/books/{id}/ - рЃгрЃўрЃњрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ
 - PUT /api/books/{id}/ - рЃгрЃўрЃњрЃюрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃцрЃџрЃЮрЃЉрЃћрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА)
 - DELETE /api/books/{id}/ - рЃгрЃўрЃњрЃюрЃўрЃА рЃгрЃљрЃерЃџрЃљ (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃцрЃџрЃЮрЃЉрЃћрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА)
 - GET /api/books/my-books/ - рЃЕрЃћрЃЏрЃў рЃгрЃўрЃњрЃюрЃћрЃЉрЃў

### рЃљрЃЋрЃбрЃЮрЃарЃћрЃЉрЃў рЃЊрЃљ рЃЪрЃљрЃюрЃарЃћрЃЉрЃў
 - GET /api/authors/ - рЃљрЃЋрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃљ
 - POST /api/authors/ - рЃљрЃЋрЃбрЃЮрЃарЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
 - GET /api/books/genres/ - рЃЪрЃљрЃюрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃљ
 - GET /api/books/conditions/ - рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃАрЃўрЃљ

### рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў
 - GET /api/requests/ - рЃЕрЃћрЃЏрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў
 - POST /api/requests/ - рЃгрЃўрЃњрЃюрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
 - GET /api/requests/incoming/ - рЃЕрЃћрЃЏрЃА рЃгрЃўрЃњрЃюрЃћрЃЉрЃќрЃћ рЃерЃћрЃЏрЃЮрЃАрЃБрЃџрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў
 - PUT /api/requests/{id}/handle/ - рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ/рЃБрЃљрЃарЃДрЃЮрЃцрЃљ

## API рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ
рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ:
 - Swagger UI: http://localhost:8000/swagger/
